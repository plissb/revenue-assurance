# Описание проекта

### Технологии:

- Spring Boot 3.4.4
- База Данных PostgrSQL 16 и ClickHouse
- Apache camel

### Описание процесса:

- Есть директория в которую попадают csv файлы.
- Апликация постоянно мониторит директорию и если там появляется фаил с маской подходящей под определение (конфигурируестя чере application.yaml) пытяется загрузить его в табицу.
- Предварительно нужно зарегитрировать папытку в таблице истории загрузок. 
  - если загрузку успешная то пометить ее как succes и указать количество загруженных rows после чего перенести фаил в дерикторию done ( конфигурируестя через application.yaml)
  - если нет то пометить как fail и перенести фаил в дерекорию errors  ( конфигурируестя через application.yaml)
- закрузить наддые в табицу data с укозанием из какого файла пришли данные.
- Пример фала днных ты docs/Transactions_20250304.csv
